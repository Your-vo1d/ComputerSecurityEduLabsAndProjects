# Метод факторизации Ферма

Этот проект реализует метод факторизации Ферма - классический алгоритм для поиска делителей чисел, основанный на представлении числа как разности квадратов.

## Описание

Код предоставляет метод `method_Ferma()` класса `Big_Number`, который выполняет факторизацию числа с использованием алгоритма Ферма.

### Алгоритм работы:

1) **Проверка на полный квадрат**: Если число является полным квадратом, сразу возвращаются два одинаковых делителя
2) **Инициализация переменных**: Устанавливаются начальные значения для x, z и y
3) **Итерационный поиск**: Алгоритм последовательно увеличивает x и проверяет, является ли z = x² - N полным квадратом
4) **Формула разности квадратов**: Если z является полным квадратом (z = y²), то N = x² - y² = (x + y)(x - y)
5) **Критерий остановки**: Поиск завершается когда x достигает значения (N + 1)/2

## Как использовать

Необходимо собрать файлы и исполняемый файл.

Параметры сборки и запуска:
```bash
g++ -std=c++11 main.cpp Big_Number.cpp -o main
./main
```

## Математическая основа

Метод основан на тождестве:
```
N = x² - y² = (x + y)(x - y)
```
Алгоритм ищет такие x и y, что разность x² - N является полным квадратом.

## Особенности алгоритма

- Эффективен для чисел, имеющих делители близкие друг к другу
- Простая и элегантная математическая основа
- Автоматически определяет тривиальный случай (полный квадрат)
- Гарантированно находит решение для нечетных составных чисел

## Требования

- Стандарт C++ 11
- Класс Big_Number с поддержкой основных арифметических операций
- Реализованные методы: `sqrt()`, арифметические операторы

## Ограничения

Метод наиболее эффективен когда два делителя числа близки по величине. Для чисел с сильно различающимися делителями производительность может снижаться.
